BUSY Bar

Boutique Téléchargement Blog

BUSY BAR est un dispositif multi-outil de productivité avec un écran LED pixelisé. Minuteur de concentration avec fonction de blocage des distractions sur votre téléphone et PC. Entièrement personnalisable, open-source et prêt pour la maison connecté

Outil de productivité

Blocage des distractions, messages BUSY personnalisé, minuteur de concentration, synchronisation multiplateforme.

Applications et intégrations

Bibliothque d'applications, connexion à des logiciels tiers, intégrations avec les événements de calendrier et les appels.

Support maison connecté

Connectez-vous à Google Home et Apple Home via le protocole Matter.

Pour les développeurs

API HTTP ouverte, SDK open-source, bibliothque PYTHON/GO/JAVASCRIPT sans verrouillage fournis.

Montrez que vous êtes occupé

Empêchez les autres d'interrompre votre concentration profonde. L'ecran affiche que vous êtes occupé et quand vous serz à nouveau disponible.

Activation instantanée

Activez et désactiver le status BUSY manuellement avec un large bouton. Fonctionne entièrement hors ligne, sans logiciel supplémentaire requis.

Set timer

Set an optional expiration timer BUSY using the scroll whell to left others known when you'll be avaible again.
